Pancakecms

A Content Management System for the modern web that uses web-components for building web pages and a server less architecture.

Website is still Under Development.

Pancake works on the concept of components. A user can add custom components that build the smallest unit on a page. Components can be arranged together to form a page template. A page template can be forked to form a page. An editor can add translations to a page and send for approval.
